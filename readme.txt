	************************************
*****  	Description de l'application:   	*****
	************************************

Agreo-Tech est une application visant à enregisrer
et afficher les parcelles, produits, employés, engins
et rendement pour un utilisateur sur votre interpretteur
de commande.




	************************************
*****  	Pré-requis:			  	*****
	************************************

Veuillez suivre les indications suivantes:

1) Installer Java JRE. Télécharger depuis le site officel d'oracle
www.oracle.com
2) lancer le logiciel et suivez les instructions




	************************************
*****  	Documentation:			  	*****
	************************************

Veuillez suivre les indications suivantes:

1) Rendez vous dans le dossier 'doc/'.

2) Double cliquez sur 'index.html'.

	
	************************************
***** 	Execution de l'application:    		*****
	************************************

Veuillez suivre les indications suivantes:
	
1) 	A l'aide de votre interpretteur de commande (cmd.exe),
	placer vous dans le dossier où vous avez téléchargé
	le fichier compressé "AGREO-TECH.jar"
	('cd [chemin]/[dossier cible]').

2)	Entrer la commande 'java -jar AGREO-TECH.jar'.
